JImageStreamToolkit is a framework to quickly process Image files.

Currently available plugins
- binarization
    - Binarize
    - OtsuBinarization
- color
    - ColorNegate
    - GrayScale
    - HistogramEqualization
- morphology
    - Dilatation
    - Erosion
- statistical
    - MedianFilter
- transformations
    - ImageCrop
    - ImageRotate
